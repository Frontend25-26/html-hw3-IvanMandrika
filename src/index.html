Квиз: Модели отображения CSS

Ответьте на вопросы, затем посмотрите правильные ответы внизу страницы.

1. Какое значение display делает элемент блочным?

2. Какие свойства входят в CSS Box Model?

3. Что делает свойство box-sizing: border-box?

4. Какие элементы по умолчанию inline?

5. Как центрировать блочный элемент по горизонтали?

Правильные ответы
Вопрос 1: block
Блочным элемент делает значение display: block.
Остальные:
  • inline делает элемент строчным;
  • flex превращает в flex-контейнер, а не просто блочный;
  • none — скрывает элемент полностью.
Вопрос 2: margin, padding, border
Box Model включает: margin (отступ), border (рамка), padding (внутренний отступ), content (содержимое). display не относится к box model!
Вопрос 3: Включает padding и border в ширину
box-sizing: border-box учитывает border и padding внутри указанной ширины элемента. Ширина не "раздувается".
Вопрос 4: <span>, <a>
<span> и <a> — это строчные (inline) элементы.
<div> и <p> — всегда блочные.
Вопрос 5: margin: 0 auto;
Для центрирования блочного элемента используется margin: 0 auto; и обязательно должна быть задана ширина.
  • padding: 0 auto; — некорректно: auto не работает для паддингов.
  • text-align: center; — работает только для центрирования текста или inline-элементов внутри блока.
